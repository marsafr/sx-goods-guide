# SX Goods Guide

Интерактивный чек-лист для команд Goods: помогает понять, что делать с изменением в подаче — обсудить с SX Goods, просто сообщить о запуске или пересобрать решение.

## Локальный запуск

```bash
npm ci
npm run dev
```

## Проверка

```bash
npm run lint
npm run build
```

Собранная версия сайта публикуется в ветке `gh-pages`.
