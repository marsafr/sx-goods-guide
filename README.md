# SX Goods Guide

Интерактивный регламент для команд Goods: помогает определить, нужно ли согласование изменения в подаче с SX Goods, и формирует список следующих действий.

## Локальный запуск

```bash
npm ci
npm run dev
```

## Проверка

```bash
npm run lint
npm run build
```

Собранная версия сайта публикуется в ветке `gh-pages`.
