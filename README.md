# SX Goods Guide

Интерактивный чек-лист для команд Goods: помогает понять, нужно ли сверить изменение в подаче с SX Goods, и формирует список следующих действий.

## Локальный запуск

```bash
npm ci
npm run dev
```

## Проверка

```bash
npm run lint
npm run build
```

Собранная версия сайта публикуется в ветке `gh-pages`.
